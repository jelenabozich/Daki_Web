    Pour demander une réservation avec le thérapeute, veuillez remplir et soumettre le formulaire ci-dessous. Le thérapeute vous contactera directement par e-mail fourni dans le formulaire pour fixer l'heure du rendez-vous.

Si vous avez des questions sur la réservation, le paiement ou toute autre chose, n'hésitez pas à utiliser le bouton Contactez-nous ci-dessous pour envoyer votre demande. Le thérapeute vous répondra dans les plus brefs délais.
